# User_behavior_analysis_system
大二的课设
用来记录大二做的一个课设实验
数据处理部分用的时spark实时处理，具体数据源找不到了，只有处理好的数据库文件。直接运行sql就行了。
后端部分是使用serlvet写的一些接口，获取数据库中的数据，并且对前端做出响应。
前端部分采用echart进行可视化，数据获取采用ajax访问后端接口获取数据
